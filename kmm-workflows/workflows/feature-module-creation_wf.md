# Feature Module Creation

> **Context**: Creating new feature. Use feature-based folders.

## Structure

```
modules/{feature}/
├── components/   # Feature UI components of the feature
├── navigation/   # Feature routes and graphs
├── ui/           # Screen composables only
└── viewModel/    # ViewModels 
```
